RANKX

Posted on 2021/08/31 | In DAX速查词典

概述

返回一个值，

迭代一参的表并计算公式后得到多个值的有序列表，然后在当前上下文中计算公式得到一个值，在有序列表中寻找该值的排名。

语法

参数 属性 描述
Table 返回表的表达式
Expression 在一参产生的行上下文和外部的上下文中计算的公式
Value 可选 在当前上下文下计算的公式，如果省略，则使用二参的Expression，通常会省略该参数
Order 可选 升序还是降序：0/FALSE/DESC表示降序；1/TRUE/ASC表示升序；默认是DESC
Ties 可选 名次是紧凑型还是跳跃型：SKIP表示跳跃型（1、1、3）；Dense表示紧凑型（1、1、2）；默认是SKIP

举例

第三参数在当前上下文中计算，没有时则使用第二参数

第三参数的使用场景

计算产品颜色在自定义的销售等级列表里面的排名

第四参数，升序或降序（默认降序）

第五参数，紧凑型或跳跃型（默认跳跃型）
